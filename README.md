- Create a website with HTML, CSS, and JavaScript.
- No frameworks or libraries allowed.
- Be creative with your design and idea.

# HTML Requirements

- [ ] Paragraph <p>
- [ ] Header <h1> - <h6>
- [ ] Footer <footer>
- [ ] strong <strong> or <b>
- [ ] emphasized <em> or <i>
- [ ] anchor ("link") <a>
- [ ] href ("link") <a href="">
- [ ] image <img>
- [ ] src ("source/location") <img src="">
- [ ] Unordered List <ul>
- [ ] List Item <li>
- [ ] blockquote <blockquote>
- [ ] Horizontal Rule <hr>
- [ ] div <div>
